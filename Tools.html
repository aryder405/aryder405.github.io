Tools

Below are links to some of the tools that we used to complete this project and a brief explanation of them.

GitHub

Used as a code repository and project management center

Isoviewer

We used this program to dig into MP4 files and see the values of their various fields. We also used this program to validate our code to make sure that it was producing the same values that were expected.

MPEG2 Packet Analyzer

This program analyzes every packet of a TS file and displays the values. This tool gave us a deeper insight into how the TS file was constructed.

MP4 ISO/IEC 14496-12

UThis is the MP4 ISO document. This document has everything you need to know about an MP4 file and by the end of this project, we had all read it front to back numerous times.

MP2TS ISO/IEC 13818-1

This is the MPEG2 TS ISO document. This document also has everything you need to know about the MPEG2 TS file format and we have all read this front to back as well.

Google Hangouts

Google hangouts is a virtual meeting platform. We used this to meet online and when we could not meet in person.
